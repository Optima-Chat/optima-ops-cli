## Installation

```bash
npm install -g @optima-chat/ops-cli
```

## Quick Start

```bash
# Service health
optima-ops services health

# Database connections
optima-ops database connections

# EC2 metrics
optima-ops infra ec2

# Recent errors
optima-ops logs errors --last 1h
```
